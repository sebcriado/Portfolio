# Steps ▶️

```
# Clone this repository
$ git clone https://github.com/Ladvace/grid-astro-portfolio
```

```
# Go into the repository
$ cd grid-astro-portfolio
```

```
# Install dependencies
$ npm install
```

```
# Start the project in development
$ npm run dev
```
